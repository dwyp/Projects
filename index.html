OSU Job Board Interface Project

This project was created to utilize a job board feed at The Ohio State University job website. Per the requirements you are able to sort and filter by job title and department. It is a full stack web application, using MySQL, Java, the Spring Framework, HTML, CSS, JavaScript, JQuery and Bootstrap.

MySQL

The MySQL database is accessed by a Spring dependency injected DAO using JDBCTemplate. It contains all the jobs ever read by the feed reader. Expired jobs are shown in their own tab on the view. The database is also used for keyword searches.

Java and Spring

The code was written in Java, driven by Agile methodologies and Test Driven Development. Tests were written for all implementations of the DAO and logic classes and a test database was created and used. The ROME tools library was used for reading the ATOM feed from OSU. Spring was used as the MVC, injecting dependancies and handling controller aspects.

HTML, CSS, JavaScript

The view was written with HTML, CSS and JavaScript. Plugins and stylesheets from JQuery and Bootstrap were used.
