Software Craftsmanship Guild Management Application

This project was created as a capstone project for the 12 week Java bootcamp at the Software Craftsmanship Guild. The application allows management and tracking of all students (potential, current and past), cohorts, hiring partners and employers and users of the application. This is a full stack application with a MySQL database, Java and Spring for the logic and controller, and HTML/CSS/JavaScript for the front end. The project uses Spring Security to handle roles and logins, dynamically changing the view depending on permissions of the user. Two users have been set up, admin and view. The admin user has all permissions, while the view user can only view information. The passwords for both users are currently set to password.

MySQL

The MySQL database contains all the data for the students, cohorts, employers and users. Tables limiting statuses and options have been created to keep the application as data driven as possible.

Java and Spring

The code was written in Java, driven by Agile methodologies and Test Driven Development. Tests were written for all implementations of the DAO and logic classes and a test database was created and used. Spring was used as the MVC, injecting dependancies and handling controller aspects. Spring Security is used for security and passwords are salted and hashed.

HTML, CSS, JavaScript

The view was written with HTML, CSS and JavaScript. Plugins and stylesheets from JQuery and Bootstrap were used.

MySQL

The MySQL database is accessed by a Spring dependency injected DAO using JDBCTemplate. It contains all the jobs ever read by the feed reader. Expired jobs are shown in their own tab on the view. The database is also used for keyword searches.

HTML, CSS, JavaScript
